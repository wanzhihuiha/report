python模块
在pycharm里模块应该就是对应java的公共类

一个模块被另一个程序第一次引入时，其主程序将运行。
如果我们想在模块被引入时，模块中的某一程序块不执行，
我们可以用__name__属性来使该程序块仅在该模块自身运行时执行。
if __name__ == '__main__':
   print('程序自身在运行')
else:
   print('我来自另一模块')
上面代码是当模块自行运行时__name__才会等于'__main__'此时说明模块不是被其他程序引用。
否则其他时候__name__是不会等于'__main__'。这样来判断模块是否是自身调用
然后__name__ 与 __main__ 底下是双下划线， _ _ 是这样去掉中间的那个空格。

使用from package import item这种形式的时候，对应的item既可以是包里面的子模块（子包），
或者包里面定义的其他名称，比如函数，类或者变量。
